Principais formatações

Nergito ou destaque

Nessa frase temos um termo em negrito usando a tag <b> (Não semantica)

Nessa frase temos um termo em destaque utilizando a tag <Strong> (semantica)

Italico ou Ênfase

Nesta frase temos um termo em italico, usando a tag <i> ou segurando Ctrl + Shift + P e Selecionando envelope de abreviação. (Não semantica)

Nesta frase temos uma ênfase em italico usando a tag <em> (Não semantica)

Texto marcado

Podemos tambem criar um texto marcado usando a tag <Mark>

Podemos tambem mudar a cor de varias marcações ao mesmo tempo utilizando o CSS

Texto grande e pequeno

Estamos criando um texto grande com a tag <big> e pequeno com a tag <small> neste paragrafo

Texto excluido

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado com a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <ins>

Existe tambem o sublinhado com a tag <u> (Não semantica)

Texto sobescrito

Para inserir coisas do tipo X20+3 com a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O com a tag <sub>
