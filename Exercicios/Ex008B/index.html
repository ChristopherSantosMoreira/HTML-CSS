Código-fonte / Pré-formatação

Abaixo você verá um codigo em JavaScript colocado utilizando a tag <code> para determinar que é um código, perceba que
a fonte ira mudar quando chegar em documents

O comando documents.getElementsById(1test') é escrito em linguagem JavaScript

Agora abaixo você vera linhas de código em python utilizando este mesmo metodo de tag

num = int (input(' Digite um número')) if num % 2 == 0: print(f'O numero {num} é PAR) else: print(f'O numero {num} é IMPAR) print('Fim do programa')

Muito feio ele inteiro, certo? Para isso colocamos a tag <pre> antes da tag <code> e ele ira seguir o padrão
que construimos o código, abaixo você vera seguindo esses passos.

    
        num = int (input(' Digite um número'))
        if num % 2 == 0:
        print(f'O numero {num} é PAR)
        else:
        print(f'O numero {num} é IMPAR)
        print('Fim do programa')

Nesta forma de cima o codigo esta com espaçamentos gigantes, para retirar isso selecionamos o código sem as
TAGS <pre> e <code> e apertamos Shift + Tab. Exemplo abaixo.

    
num = int (input(' Digite um número'))
if num % 2 == 0:
print(f'O numero {num} é PAR)
else:
print(f'O numero {num} é IMPAR)
print('Fim do programa')

Pronto, agora ele está de forma mais facil para o entedimento de quem estiver lendo, com espaçamentos e dividido em linhas da forma correta!
